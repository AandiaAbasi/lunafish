## انواع سوالات

هر نوع سوال به روش مختلفی جواب می‌خواهد.

### 1️⃣ تکمیلی (Completion)

**سوال نمونه:**
```
کلمه _____ به معنی سلام است.
```

**چگونه جواب بدهید:**
```
☆ جملو را تکمیل کنید
☆ جاهای خالی را پر کنید

مثال:
کلمه "Hello" به معنی سلام است.
```

**نکات:**
- ✅ دقیق تایپ کنید
- ✅ تمام جاهای خالی را پر کنید
- ❌ اضافه‌تر یا کم‌تر نویسید

---

### 2️⃣ چند گزینه‌ای (Multiple Choice)

**سوال نمونه:**
```
پایتخت فرانسه کجاست؟

☐ لندن
☐ پاریس
☐ رم
☐ برلین
```

**چگونه جواب بدهید:**
```
☆ یک گزینه را انتخاب کنید
☆ فقط یکی را علامت بزنید

مثال:
☑ پاریس
```

**نکات:**
- ✅ تمام گزینه‌ها را بخوانید
- ✅ بهترین جواب را انتخاب کنید
- ❌ بیش از یک گزینه انتخاب نکنید

---

### 3️⃣ درست/غلط (True/False)

**سوال نمونه:**
```
پاریس پایتخت انگلستان است.

☐ درست
☐ غلط
```

**چگونه جواب بدهید:**
```
☆ "درست" یا "غلط" را انتخاب کنید
☆ تنها یکی را انتخاب کنید

مثال:
☑ غلط
```

**نکات:**
- ✅ سوال را دقیق بخوانید
- ✅ فقط یکی انتخاب کنید
- ❌ هر دوی را انتخاب نکنید

---

### 4️⃣ جفت‌کردن (Matching)

**سوال نمونه:**
```
اصطلاح انگلیسی را با معنی فارسی جفت کنید:

Apple          1. موز
Banana         2. سیب
Orange         3. پرتقال
```

**چگونه جواب بدهید:**
```
☆ هر اصطلاح را با تعریف صحیح جفت کنید

مثال:
Apple      → 2. سیب
Banana     → 1. موز
Orange     → 3. پرتقال
```

**نکات:**
- ✅ تمام اصطلاح‌ها را جفت کنید
- ✅ هر تعریف یک بار استفاده شود
- ❌ تعریفی را دوبار استفاده نکنید

---

### 5️⃣ جای‌خالی (Fill in the Blank)

**سوال نمونه:**
```
سرزمین _____ و _____ برای تفریح معروف است.
```

**چگونه جواب بدهید:**
```
☆ جاهای خالی را پر کنید

مثال:
سرزمین "سوئیس" و "اتریش" برای تفریح معروف است.
```

**نکات:**
- ✅ بترتیب پر کنید
- ✅ تمام جاهای خالی را پر کنید
- ❌ فقط برخی را پر نکنید

---

### 6️⃣ جواب کوتاه (Short Answer)

**سوال نمونه:**
```
پایتخت ایران کجاست؟
```

**چگونه جواب بدهید:**
```
☆ جواب مختصر و درست بنویسید

مثال:
"تهران" یا "شهر تهران"
```

**نکات:**
- ✅ جواب را کامل بنویسید
- ✅ واضح و روشن باشد
- ❌ خلاصه خیلی کوتاه نباشد

```

```

```

```


## مشاهده نتایج

### بلافاصل بعد از ارسال

```
سوالات خودکار:
- چند گزینه‌ای: ✅ بلافاصل نمره شد
- درست/غلط: ✅ بلافاصل نمره شد

سوالات دستی:
- تکمیلی: ⏳ منتظر نمره معلم
- جواب کوتاه: ⏳ منتظر نمره معلم
- جفت‌کردن: ⏳ منتظر نمره معلم
```

### بعد از نمره دهی معلم

```
✅ تمام سوالات نمره شدند
📊 نتیجه نهایی نمایش داده می‌شود
```

### دیدن نتایج

```
روی "نتایج" کلیک کنید

نتایج شما:
- تمرین 1: 85/100
- تمرین 2: 90/100
- تمرین 3: (منتظر نمره)
```

## زمان تمرین

### مدت زمان

```
معلم زمان مشخصی برای تمرین تعیین کرد:
- مثلاً: 30 دقیقه

هنگام شروع، تایمر شروع می‌شود.
```

### اگر زمان تمام شود

```
⏰ زمان تمام شد!

جواب‌های شما:
✅ ارسال می‌شوند
📊 نمره می‌شوند
```

### نکات مهم

```
☆ زمان خود را مدیریت کنید
☆ شتاب نزنید
☆ عجله نکنید
☆ جواب‌ها را دوباره بررسی کنید
```

---

## نکات مهم

### ✅ موارد خوب

1. **دقت** - سوال را با دقت بخوانید
2. **زمان** - زمان خود را مدیریت کنید
3. **پاکی** - نوشتار خود را تمیز و واضح نگاه دارید
4. **سؤال‌پرسی** - اگر سوالی نفهمیدید بپرسید
5. **یادگیری** - نتایج را ببینید و یاد بگیرید

### ❌ موارد بد

1. ❌ **تقلب** - جواب دوستتان را کپی نکنید
2. ❌ **عجله** - سریع و بی‌دقت جواب ندهید
3. ❌ **ترک** - تمرین را نیمه‌کاره ترک نکنید
4. ❌ **دروغ** - جواب غلط برای خودتان خوب نیست
5. ❌ **بی‌اعتنایی** - تمرین را جدی بگیرید

## سوالات متداول

### س: می‌توانم تمرین را چند بار انجام دهم؟

**ج:** بله! می‌توانید چند بار تلاش کنید. بهترین امتیاز شما محفوظ می‌شود.

---

### س: اگر جواب را غلط بدهم؟

**ج:** نگران نباشید!
- ❌ جواب غلط = یادگیری
- ✅ بعد بهتر می‌فهمید
- ✅ می‌توانید دوباره تلاش کنید

---

### س: چگونه می‌توانم جواب‌های قبلی را ببینم؟

**ج:** 
```
نتایج → تمرین‌های قبلی
```

تمام تمرین‌های شما نمایش داده می‌شوند.

---

### س: پس از ارسال می‌توانم تمرین را ویرایش کنم؟

**ج:** خیر، بعد از ارسال نمی‌توانید تغییر دهید.

---

### س: معلم کی نتیجه را می‌دهد؟

**ج:** معمولاً:
- سوالات خودکار: فوری
- سوالات دستی: 24-48 ساعت

---

### س: اگر سیستم خراب شود؟

**ج:** 
- معلم را خبر دهید
- دوباره تلاش کنید
- نترس، تمرین شما ذخیره شده است

---

### س: تیمر خودکار متوقف می‌شود؟

**ج:** نه، تیمر همیشه جاری است.
- اگر برنامه بسته شود: ⏸ موقت می‌شود
- اگر دوباره باز شود: ⏯ ادامه می‌دهد
