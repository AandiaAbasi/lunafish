# 📚 راهنمای سیستم آزمون برای دانشآموزان

سلام! این راهنما شما را در حل آزمون‌های درسی اپ فوفوفیش راهنمایی می‌کند.

---

## 🎯 شروع سریع (5 دقیقه)

### مرحله 1️⃣: آزمون را باز کنید
1. به بخش "کلاس‌های من" بروید
2. کلاس مورد نظر را انتخاب کنید
3. روی "آزمون" کلیک کنید

### مرحله 2️⃣: سؤالات را ببینید
- تمام سؤالات برای شما نمایش داده می‌شود
- هر سؤال می‌تواند تصویر، صوت یا ویدیو داشته باشد
- راهنمایی برای هر سؤال موجود است

### مرحله 3️⃣: پاسخ دهید

**برای سؤالات تایپی:**
- متن خود را در کادر مربوطه وارد کنید

**برای سؤالات چند گزینه‌ای:**
- روی یکی از گزینه‌ها کلیک کنید

**برای سؤالات تک گزینه‌ای:**
- فقط یک گزینه را انتخاب کنید

### مرحله 4️⃣: ارسال کنید
1. بعد از پاسخ دادن به تمام سؤالات، روی "ارسال" کلیک کنید
2. نمره شما فوری محاسبه می‌شود
3. نتیجه فوری نمایش داده می‌شود

---

## 📝 انواع سؤالات

### ✏️ سؤالات تایپی (تایپی)
شما می‌بایست متن پاسخ را تایپ کنید.

**مثال:**
```
سؤال: نام پایتختِ فرانسه چیست؟
پاسخ: پاریس
```

### 🔘 سؤالات تک گزینه‌ای (RadioButton)
فقط یکی از گزینه‌ها را انتخاب کنید.

**مثال:**
```
سؤال: دو به اضافه دو برابر است با:
گزینه‌ها:
  ⭕ 3
  ⭕ 4 (صحیح)
  ⭕ 5
```

### ☑️ سؤالات چند گزینه‌ای (Checkbox)
می‌توانید چند گزینه را انتخاب کنید.

**مثال:**
```
سؤال: کدام کشورها در اروپا هستند؟
گزینه‌ها:
  ☑ فرانسه (صحیح)
  ☑ آلمان (صحیح)
  ☐ استرالیا
```

---

## 📊 نمره‌گذاری

### چطور نمره محاسبه می‌شود؟

```
✅ پاسخ صحیح = 1 امتیاز
❌ پاسخ غلط = 0 امتیاز
```

**مثال:**
- 3 سؤال در آزمون
- 2 پاسخ صحیح ✅
- 1 پاسخ غلط ❌
- نمره شما: 2/3 = **66.67%**

## 🎯 نکات مهم

### ✅ قبل از ارسال
- [ ] تمام سؤالات را پاسخ دادید؟
- [ ] پاسخ‌های خود را چک کردید؟
- [ ] مطمئن هستید که می‌خواهید ارسال کنید؟

### ⚠️ بعد از ارسال
- نمی‌تواند پاسخ‌ها را تغییر دهید
- ولی می‌توانید آزمون را دوباره حل کنید
- نمره‌های قبلی ذخیره می‌شود

---

## 📱 نتایج

### نتایج خود را ببینید
1. روی "نتایج" کلیک کنید
2. تمام تلاش‌های شما نمایش داده می‌شود
3. روی هر تلاش برای دیدن جزئیات کلیک کنید

### جزئیات هر تلاش
- تمام سؤالات
- پاسخ‌های شما
- پاسخ‌های صحیح ✅
- نمره هر سؤال
- نمره کل و درصد

---

## 💡 نکات و ترفندها

### 🎯 برای بهتر شدن
1. **آهسته بخوانید** - هر سؤال را به دقت بخوانید
2. **راهنمایی استفاده کنید** - اگر راهنمایی موجود بود، بخوانید
3. **تصویر‌ها را ببینید** - اگر تصویر داشتند، توجه کنید
4. **دوباره حل کنید** - نتایج قدیم ذخیره می‌شود

### ⚡ نکات سریع
- نمره فوری محاسبه می‌شود
- تلاش‌های قدیم حفظ می‌شود
- می‌توانید بارها حل کنید
- نتایج همیشه موجود است

## ❓ سؤالات متکرر

### Q: پاسخ اشتباه را دادم، چطور تصحیح کنم؟
**A:** بعد از ارسال نمی‌توانید تغییر دهید. ولی می‌توانید آزمون را دوباره حل کنید و نمره قدیم ذخیره می‌شود.

### Q: زمان محدود برای پاسخ دادن دارم؟
**A:** بستگی به معلم دارد. اگر وقت محدود بود، تایمر نمایش داده می‌شود.

### Q: نمره شما چگونه محاسبه می‌شود؟
**A:** تعداد پاسخ‌های صحیح تقسیم بر کل سؤالات ضرب در 100 = درصد

### Q: آیا می‌توانم نتایج قدیم را ببینم؟
**A:** بله! روی "نتایج" کلیک کنید و تمام تلاش‌های قدیم را ببینید.

### Q: چه کنم اگر اینترنت قطع شود؟
**A:** آپ پاسخ شما را به خاطر می‌سپارد. وقتی اینترنت برگردد، دوباره امتحان کنید.

## 🔌 چطور داده‌ها منتقل می‌شوند؟

### Flow حل آزمون

وقتی شما آزمون را حل می‌کنید:

```
1. شما ⬅️ اپ موبایل
   (پاسخ‌ها را انتخاب می‌کنید)
        ⬇️
2. اپ ➜ سرور Backend  
   (POST /api/exercise/exam/5/submit/)
        ⬇️
3. سرور محاسبه می‌کند:
   ✓ کدام پاسخ صحیح است
   ✓ کدام پاسخ غلط است
   ✓ نمره کل
        ⬇️
4. سرور ⬅️ اپ
   (نتیجه و نمره)
```

### مثال: ارسال پاسخ‌ها

**شما این کار را می‌کنید:**
```
✓ سؤال 1: "پاریس" را انتخاب کنید
✓ سؤال 2: "علی" را بنویسید
✓ سؤال 3: "فرانسه" و "آلمان" را انتخاب کنید
✓ دکمه ارسال را بزنید
```

**اپ این را می‌فرستد:**
```json
{
  "teachingsubject_id": 5,
  "answers": [
    {"field_id": 1, "field_detail_id": 2},
    {"field_id": 2, "value": "علی"},
    {"field_id": 3, "field_detail_id": 10},
    {"field_id": 3, "field_detail_id": 11}
  ]
}
```

**سرور محاسبه می‌کند:**
```
سؤال 1: 
  • شما انتخاب کردید: "پاریس" (گزینه 2)
  • صحیح است ✓
  • نمره: 1 امتیاز

سؤال 2:
  • شما نوشتید: "علی"
  • معلم بعداً بررسی می‌کند
  • نمره: منتظر تایید

سؤال 3:
  • شما انتخاب کردید: "فرانسه" و "آلمان"
  • هر دو صحیح است ✓
  • نمره: 1 امتیاز

نتیجه نهایی:
  2 صحیح + 0 منتظر = 2/3 = 66.67%
```

**سرور جواب می‌دهد:**
```json
{
  "data": {
    "id": 42,
    "score": 2,
    "correct": 2,
    "incorrect": 1,
    "total_questions": 3,
    "percentage": 66.67,
    "details": [
      {
        "field": 1,
        "field_title": "پایتخت فرانسه کجاست؟",
        "option_title": "پاریس",
        "score": 1
      },
      {
        "field": 2,
        "field_title": "نام شما؟",
        "value": "علی",
        "score": 0
      },
      {
        "field": 3,
        "field_title": "کدام کشورهای اروپایی هستند؟",
        "option_title": "فرانسه",
        "score": 1
      }
    ]
  },
  "message": "پاسخ‌ها با موفقیت ثبت شدند"
}
```

```
POST /api/exercise/exam/{subject_id}/submit/
```

```json
  "answers": [

    {
```

**پاسخ 201:**
```json

    "subject_title": "ریاضی 1",

    "correct": 2,
    "incorrect": 1,
    "total_questions": 3,
    "percentage": 66.67,
    "details": [
      {
        "field_id": 1,
        "field_title": "دو به اضافه دو برابر است با:",
        "option_title": "4",
        "score": 1
      },
      {
        "field_id": 2,
        "field_title": "نام خود را بنویسید",
        "value": "علی حسینی",
        "score": 0
      }
    ]
  },
  "message": "پاسخ‌ها با موفقیت ثبت شدند"

```

**کد نمونه React Native:**
```javascript
const submitExam = async (subjectId, answers) => {
  try {
    setLoading(true);

    const response = await fetch(
      `http://api.example.com/api/exercise/exam/${subjectId}/submit/`,
      {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${accessToken}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          teachingsubject_id: subjectId,
          answers: answers
        })
      }
    );

    const result = await response.json();
    if (!response.ok) throw new Error(result.error);

    // نمایش نتیجه
    showScore(result.data.score, result.data.total_questions);
    
    // ذخیره برای نمایش بعدی
    setLastAttempt(result.data);
    navigation.navigate('Results', { attemptId: result.data.id });
    
  } catch (error) {
    console.error('Error:', error);
    showError(error.message);
  } finally {
    setLoading(false);
  }
};
```
