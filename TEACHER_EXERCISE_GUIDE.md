## شروع سریع

### مرحله 1: وارد شوید

```
http://localhost:8000/admin/
- نام کاربری: نام کاربری شما
- رمز عبور: رمز عبور شما
```

### مرحله 2: به قسمت Exercise بروید

```
Admin → Exercise → Exercises → Add Exercise
```

### مرحله 3: اطلاعات تمرین را وارد کنید

- **عنوان تمرین**: "تمرین شماره 1 - الفبا"
- **موضوع**: درسی که تدریس می‌کنید را انتخاب کنید
- **نوع تمرین**: 
  - تمرین معمولی (practice)
  - تکلیف خانگی (homework)
  - کوئیز (quiz)
  - آزمون (exam)
- **سطح سختی**: آسان / متوسط / سخت
- **مدت زمان**: چند دقیقه طول می‌کشد؟
- **کل امتیاز**: کل نمره (مثال: 100)

### مرحله 4: منتشر کنید

```
☑ منتشر شده
```

## ایجاد تمرین

### معلومات پایه تمرین

| فیلد | توضیح | مثال |
|------|-------|------|
| عنوان | نام تمرین | تمرین شماره 1 - الفبا |
| توضیح | شرح تفصیلی | در این تمرین دانش‌آموزان الفبای انگلیسی را یاد می‌گیرند |
| موضوع | درس | انگلیسی - سطح 1 |
| معلم | شما خود (خودکار) | علی محمدی |

### تنظیمات تمرین

#### نوع تمرین
- **تمرین معمولی**: تمرین عادی در کلاس یا خانه
- **تکلیف خانگی**: تمرین برای خانه
- **کوئیز**: آزمون کوتاه
- **آزمون**: آزمون نهایی

#### سطح سختی
- **آسان** (easy): برای دانش‌آموزان ضعیف
- **متوسط** (medium): برای اکثر دانش‌آموزان
- **سخت** (hard): برای دانش‌آموزان خوب

#### مدت زمان
```
مثال‌ها:
- تمرین ساده: 10-15 دقیقه
- تمرین معمولی: 20-30 دقیقه
- آزمون: 45-90 دقیقه
```

#### نمره قبولی
```
معمولاً:
- 60% برای تمرین معمولی
- 70% برای تکلیف خانگی
- 80% برای آزمون
```

### تمرین‌های اختیاری

تیک کنید اگر:
- این تمرین اختیاری است
- دانش‌آموزان انجام دادن آن لازم نیست
- فقط برای دانش‌آموزان علاقه‌مند است

## اضافه کردن سوالات

### مرحله 1: پس از ذخیره تمرین

```
Exercise → تمرین خود → Questions → Add Question
```

### مرحله 2: انتخاب نوع سوال

```
نوع سوال:
☐ تکمیلی (Completion)
☐ چند گزینه‌ای (Multiple Choice)
☐ درست/غلط (True/False)
☐ جفت‌کردن (Matching)
☐ جای‌خالی (Fill in the Blank)
☐ جواب کوتاه (Short Answer)
```

### مرحله 3: متن سوال را وارد کنید

```
متن سوال: "الفبای انگلیسی از چند حرف تشکیل شده است؟"
```

### مرحله 4: ذخیره کنید

تمام سوالات خودکار به این تمرین اضافه می‌شوند.

## انواع سوالات

### 1️⃣ تکمیلی (Completion)

دانش‌آموز باید جملو را تکمیل کند.

**مثال:**
```
سوال: "کلمه _____ به معنی سلام است."
جواب: "Hello"
```

**چگونه اضافه کنید:**
1. نوع: Completion
2. متن: "کلمه _____ به معنی سلام است."
3. جواب: "Hello"
4. ذخیره

---

### 2️⃣ چند گزینه‌ای (Multiple Choice)

دانش‌آموز یک گزینه را انتخاب می‌کند.

**مثال:**
```
سوال: "پایتخت فرانسه کجاست؟"
گزینه‌ها:
☐ لندن
☑ پاریس
☐ رم
☐ برلین
```

**چگونه اضافه کنید:**
1. نوع: Multiple Choice
2. متن: "پایتخت فرانسه کجاست؟"
3. گزینه‌ها:
   - لندن (غلط)
   - پاریس (✓ صحیح)
   - رم (غلط)
   - برلین (غلط)
4. ذخیره

---

### 3️⃣ درست/غلط (True/False)

دانش‌آموز درست یا غلط را انتخاب می‌کند.

**مثال:**
```
سوال: "پاریس پایتخت انگلستان است."
☐ درست
☑ غلط
```

**چگونه اضافه کنید:**
1. نوع: True/False
2. متن: "پاریس پایتخت انگلستان است."
3. جواب صحیح: غلط
4. ذخیره

---

### 4️⃣ جفت‌کردن (Matching)

دانش‌آموز اشیاء را با یکدیگر جفت می‌کند.

**مثال:**
```
جفت کنید:
Apple     →  سیب
Banana    →  موز
Orange    →  پرتقال
```

**چگونه اضافه کنید:**
1. نوع: Matching
2. اصطلاح‌های اول: "Apple", "Banana", "Orange"
3. تعریف‌ها: "سیب", "موز", "پرتقال"
4. جفت‌کنید
5. ذخیره

---

### 5️⃣ جای‌خالی (Fill in the Blank)

دانش‌آموز جاهای خالی را پر می‌کند.

**مثال:**
```
سوال: "سرزمین _____ و _____ برای تفریح معروف است."
جواب: "سوئیس", "اتریش"
```

**چگونه اضافه کنید:**
1. نوع: Fill in the Blank
2. متن: "سرزمین _____ و _____ برای تفریح معروف است."
3. جواب‌ها: "سوئیس", "اتریش"
4. ذخیره

---

### 6️⃣ جواب کوتاه (Short Answer)

دانش‌آموز به اختصار جواب می‌دهد.

**مثال:**
```
سوال: "پایتخت ایران کجاست؟"
جواب: "تهران"
```

**چگونه اضافه کنید:**
1. نوع: Short Answer
2. متن: "پایتخت ایران کجاست؟"
3. جواب انتظار: "تهران"
4. ذخیره

## گروه‌های انتخاب

دانش‌آموزان می‌توانند بین چندین تمرین انتخاب کنند.

### مثال 1: انتخاب سطح سختی

```
"تمرین‌های انگلیسی - سطح سختی را انتخاب کنید"

☐ آسان
☐ متوسط
☑ سخت
```

### مثال 2: انتخاب موضوع

```
"موضوع را انتخاب کنید"

☐ الفبا
☑ اعداد
☐ روزهای هفته
```

### چگونه ایجاد کنید

#### مرحله 1: ایجاد تمرین‌های مختلف

```
1. تمرین الفبا - آسان
2. تمرین الفبا - متوسط
3. تمرین الفبا - سخت
```

#### مرحله 2: ایجاد گروه انتخاب

```
Admin → Exercise Choices → Add Exercise Choice

- عنوان: "تمرین‌های انگلیسی - سطح سختی را انتخاب کنید"
- نوع: difficulty (سطح سختی)
- تمرین‌ها: 3 تمرین بالا را انتخاب کنید
- تعداد انتخاب: 1 (دانش‌آموز یکی انتخاب می‌کند)
- منتشر شده: ☑
```

#### مرحله 3: منتظر انتخاب دانش‌آموزان

دانش‌آموزان خود سطح را انتخاب می‌کنند.

## مشاهده نتایج دانش‌آموزان

### مرحله 1: به Student Exercise Attempts بروید

```
Admin → Exercise → Student Exercise Attempts
```

### چه اطلاعاتی می‌بینید؟

| ستون | معنی |
|------|------|
| دانش‌آموز | نام دانش‌آموز |
| تمرین | نام تمرین |
| وضعیت | شروع شده / ارسال شده / نمره شده |
| امتیاز | امتیاز دریافت شده |
| زمان | وقتی شروع کرد |

### مرحله 2: جزئیات یک تمرین را ببینید

```
روی یک سطر کلیک کنید → جواب‌های دانش‌آموز دیده می‌شود
```

### مرحله 3: نمره دهی

```
- جواب‌های خودکار نمره شده‌اند (چند گزینه‌ای، درست/غلط)
- جواب‌های کوتاه نیاز به نمره دستی دارند
```

---

## نمره دهی دستی

### جواب‌هایی که نیاز به نمره دستی دارند

- ✓ تکمیلی
- ✓ جای‌خالی
- ✓ جواب کوتاه
- ✗ چند گزینه‌ای (خودکار)
- ✗ درست/غلط (خودکار)

### چگونه نمره بدهید

```
1. به جواب کوتاه روی کلیک کنید
2. "نمره دهی" را انتخاب کنید
3. "صحیح" یا "غلط" را انتخاب کنید
4. ذخیره کنید
```

---

## نکات مهم

### ✅ موارد مهم

1. **تمرین را منتشر کنید** - وگرنه دانش‌آموزان نمی‌بینند
2. **سوالات واضح باشند** - دانش‌آموزان باید درک کنند
3. **گزینه‌های اضافی اضافه کنید** - حداقل 3-4 گزینه
4. **زمان کافی** - دانش‌آموزان فشار نخورند
5. **نمره منصفانه** - با سطح سختی مطابقت داشته باشد

### ❌ موارد غلط

- سوال ابهام‌دار
- جواب چند برابر
- گزینه‌های تقریبی
- نمره بیش از حد
- تمرین سخت‌تر از توضیح

---

## سوالات متداول

### س: دانش‌آموزان می‌توانند چند بار تلاش کنند؟

**ج:** بله، می‌توانند تمرین را چند بار انجام دهند. شما بهترین امتیاز را می‌بینید.

---

### س: اگر دانش‌آموز سوالی را نفهمید؟

**ج:** می‌توانید توضیح اضافی در توضیحات تمرین اضافه کنید.

---

### س: چگونه تمرین را کپی کنم؟

**ج:** اگر تمرین برای کلاس دوم هم می‌خواهید:
1. تمرین را باز کنید
2. "SaveAs" یا "کپی" را انتخاب کنید
3. برای کلاس دوم تنظیم کنید

---

### س: دانش‌آموز چند وقت برای انجام تمرین دارد؟

**ج:** مدت زمانی که تنظیم کردید (مثلاً 30 دقیقه). پس از آن دیگر نمی‌توانند جواب بدهند.

---

## مثال کامل: تمرین ایجاد کنید

### مرحله 1: تمرین‌ها ایجاد کنید

```
1️⃣ تمرین: "اعداد 1-10"
   - نوع: تمرین معمولی
   - سطح: آسان
   - مدت: 15 دقیقه
   - امتیاز: 50

2️⃣ تمرین: "اعداد 1-100"
   - نوع: تمرین معمولی
   - سطح: متوسط
   - مدت: 20 دقیقه
   - امتیاز: 100

3️⃣ تمرین: "اعداد پیچیده"
   - نوع: تمرین معمولی
   - سطح: سخت
   - مدت: 30 دقیقه
   - امتیاز: 150
```

### مرحله 2: سوالات اضافه کنید

```
سوال 1: "عدد 5 به انگلیسی چگونه است؟"
نوع: چند گزینه‌ای
گزینه‌ها: Four, Five, Six, Seven

سوال 2: "عدد 10 به انگلیسی درست است؟"
نوع: درست/غلط
جواب: درست
```

### مرحله 3: گروه انتخاب ایجاد کنید

```
عنوان: "اعداد - سطح خود را انتخاب کنید"
نوع: difficulty
تمرین‌ها: 3 تمرین بالا
منتشر شده: ☑
```

### مرحله 4: منتظر دانش‌آموزان

- دانش‌آموزان سطح را انتخاب می‌کنند
- تمرین را انجام می‌دهند
- شما نتایج را می‌بینید
