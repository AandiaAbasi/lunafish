## ثبت‌نام و ورود

### مرحله 1: ثبت‌نام معلم

1. **باز کردن اپلیکیشن یا وب‌سایت**
   - به آدرس پلتفرم Luna Fish بروید

2. **انتخاب "ثبت‌نام معلم"**
   - روی دکمه "ثبت‌نام به عنوان معلم" کلیک کنید

3. **وارد کردن شماره تلفن یا ایمیل**
   - شماره تلفنی (09xxxxxxxxx) یا ایمیل خود را وارد کنید

4. **دریافت و وارد کردن کد OTP**
   - کد 6 رقمی دریافت کنید
   - کد را وارد کنید

5. **تکمیل اطلاعات حساب**
   - نام کاربری و رمز عبور انتخاب کنید
   - نام کامل خود را وارد کنید
   - تخصص/حوزه تدریس را مشخص کنید (مثال: "English for Kids")
   - سابقه تدریس (سال‌ها) را وارد کنید
   - (اختیاری) ویدیو معرفی اضافه کنید

6. **تأیید حساب**
   - معلم یا مدیر سامانه حساب شما را تأیید می‌کند
   - شما ایمیل تأیید دریافت می‌کنید

### مرحله 2: ورود

```
POST /api/teacher/send-otp/
{
  "identifier": "09123456789"  // یا ایمیل
}
```

```
POST /api/teacher/verify-otp/
{
  "identifier": "09123456789",
  "code": "123456"
}
```

## مدیریت کلاس‌ها

### ایجاد کلاس جدید

1. **روی "کلاس جدید" کلیک کنید**

2. **تعیین اطلاعات کلاس:**
   - **نام کلاس:** (مثال: "English Level 1 - Group A")
   - **تاریخ و ساعت:** وقت شروع کلاس
   - **مدت زمان:** (مثال: 60 دقیقه)
   - **تعداد دانش‌آموز:** حداکثر دانش‌آموزین
   - **سطح زبانی:** (مبتدی، متوسط، پیشرفته)
   - **توصیف:** توضیح درس امروز

3. **اضافه کردن دانش‌آموزان:**
   - دانش‌آموزان را از لیست انتخاب کنید
   - یا آن‌ها را دعوت کنید

4. **ذخیره و انتشار:**
   - روی "ذخیره" کلیک کنید
   - دانش‌آموزان آن را می‌بینند

### ویرایش و حذف کلاس

- روی کلاس کلیک کنید
- روی "ویرایش" یا "حذف" کلیک کنید
- تغییرات را تأیید کنید

## ابزارهای تدریس

### 🎥 ابزارهای ویدیویی و صوتی

#### کنترل دوربین و میکروفون
```
- دوربین خود را روشن/خاموش کنید
- دوربین دانش‌آموز را کنترل کنید (روشن/خاموش)
- میکروفون را مدیریت کنید
```

#### بررسی کیفیت اتصال
- نماد شبکه را ببینید (سبز = خوب، نارنجی = متوسط، قرمز = ضعیف)
- اگر دانش‌آموز اتصال ضعیف دارد، می‌توانید او را درباره آن مطلع کنید

#### سوئیچ دوربین (برای دستگاه‌های موبایل)
- بین دوربین جلو و عقب تعویض کنید
- برای نشان دادن چیزهای فیزیکی مفید است

### 📝 تخته سفید (Whiteboard)

#### ابزارهای رسم
- **مداد رنگی:** برای نوشتن و رسم
- **پاک‌کن:** حذف بخش‌های اشتباه
- **متن:** اضافه کردن متن تایپ شده
- **اشکال:** رسم اشکال هندسی (دایره، مربع، مثلث)

#### ویژگی‌های پیشرفته
- **Undo/Redo:** بازگردان یا تکرار اقدامات
- **بزرگ‌نمایی:** برای دیدن بهتر جزئیات
- **بارگذاری تصویر:** آپلود تصویر برای تدریس
- **رنگ‌بندی:** انتخاب رنگ‌های مختلف

#### قفل صفحه
```
- روی "قفل صفحه" کلیک کنید
- فقط شما می‌توانید بنویسید
- دانش‌آموزان فقط می‌بینند
```

### 💬 نوار ابزار تعامل سریع

#### دکمه "دست بالا"
- دانش‌آموز روی این دکمه کلیک می‌کند
- شما می‌بینید که کی دست بالا کرده است
- او را برای سخن گفتن انتخاب کنید

#### ارسال استیکر و واکنش
- 👏 **تشویق:** برای تشویق دانش‌آموز
- 👍 **لایک:** برای تایید پاسخ درست
- ⭐ **ستاره:** برای عملکرد عالی
- 😊 **ایموجی:** احساسات مختلف

#### چت سریع
- پیام‌های متنی برای کلاس
- ایموجی و نماد‌های خاص
- پیام‌های خصوصی به دانش‌آموز واحد

#### "مکث کلاس"
```
- اگر اضطراری پیش آمد
- روی "مکث کلاس" کلیک کنید
- کلاس موقتاً متوقف می‌شود
- بعدها دوباره آن را شروع کنید
```

### 🎯 مدیریت دانش‌آموزان در کلاس

#### بی‌صدا کردن
```
- روی دکمه میکروفون دانش‌آموز کلیک کنید
- او نمی‌تواند صحبت کند (اما می‌بیند)
- برای نظم کلاس مفید است
```

#### اخراج از کلاس
```
- موقتی: دانش‌آموز می‌تواند دوباره وارد شود
- دائم: دانش‌آموز نمی‌تواند دوباره وارد شود
```

### ⏰ تایمر برای مدیریت زمان

```
- روی "تایمر" کلیک کنید
- زمان را تنظیم کنید (مثال: 10 دقیقه)
- برای فعالیت‌های زمان‌محدود استفاده کنید
```

### 📋 حضور و غیاب

#### نحوه ثبت
- **خودکار:** سیستم خودکار فعالیت دانش‌آموز را ثبت می‌کند
- **دستی:** شما می‌توانید دستی علامت بزنید

#### نظارت
- لیست دانش‌آموزان و وضعیت حضور‌شان را ببینید
- یادداشت‌های خصوصی برای هر دانش‌آموز

## مدیریت دانش‌آموزان

### 📚 لیست دانش‌آموزان

```
داشبورد → دانش‌آموزان
- نام دانش‌آموز
- سطح زبانی
- تعداد کلاس‌ها
- امتیاز کسب شده
- وضعیت (فعال/غیرفعال)
```

### یادداشت‌برداری

برای هر دانش‌آموز بنویسید:
- نقاط قوت
- نقاط ضعف
- توصیه‌های بهبود
- تصرفات خاص

### ارسال فایل و محتوا

```
- روی "ارسال فایل" کلیک کنید
- PDF یا سند را انتخاب کنید
- دانش‌آموزان آن را می‌بینند و دانلود می‌کنند
```

### ضبط جلسه

```
- قبل از شروع کلاس، "ضبط جلسه" را فعال کنید
- بعد از پایان، فایل ویدیو ذخیره می‌شود
- دانش‌آموزان غایب می‌توانند آن را ببینند
```

## کوییز و آزمون‌ها

### طراحی کوییز لحظه‌ای

```
1. در طول کلاس، روی "کوییز جدید" کلیک کنید
2. سوال را وارد کنید:
   - گزینه‌های مختلف
   - پاسخ صحیح را مشخص کنید
3. دانش‌آموزان 30 ثانیه برای پاسخ دارند
4. نتایج فوری نشان داده می‌شوند
5. پاسخ‌های درست امتیاز می‌گیرند
```

### مثال: کوییز ساده

**سوال:** Which word means "happy"?
- A) Sad
- B) Happy ✓
- C) Angry

## نکات و ترفندها

### 💡 بهترین روش‌های تدریس

✅ **شروع به موقع**
- 5 دقیقه قبل سیستم را بررسی کنید

✅ **محتوای آماده**
- تخته سفید را از قبل آماده کنید
- فایل‌ها را آپلود کنید

✅ **تعامل بیشتر**
- کوییز‌های سریع بپرسید
- دانش‌آموزان را شرکت دهید

✅ **کنترل کلاس**
- قانون‌های روشنی تعیین کنید
- از بی‌صدا کردن استفاده کنید

✅ **ضبط کنید**
- جلسات را ضبط کنید
- دانش‌آموزان غایب می‌توانند ببینند

### 🚀 ترفندهای پیشرفته

**تخته سفید موثر:**
- تصویرها کوچک نباشند
- رنگ‌های متفاوت استفاده کنید
- خط‌ها روشن و خوانا باشند

**کنترل بهتر:**
- دانش‌آموزانی را که حواسشان پرت است بی‌صدا کنید
- با آن‌ها خصوصی صحبت کنید

**امتیازدهی دقیق:**
- شرکت فعال را پاداش دهید
- نه فقط پاسخ‌های درست
