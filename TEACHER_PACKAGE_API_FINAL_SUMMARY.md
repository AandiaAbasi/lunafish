## 🔧 Quick Command Reference

```bash
# Create a package
curl -X POST http://localhost:8000/api/teacher/packages/ \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"Course","total_sessions":10,"total_price":"1000000"}'

# List packages
curl http://localhost:8000/api/teacher/packages/ \
  -H "Authorization: Bearer $TOKEN"

# Add installment
curl -X POST http://localhost:8000/api/teacher/packages/{id}/installments/ \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"session_number":5,"amount":"300000"}'

# See more examples in TEACHER_PACKAGE_API_CHEAT_SHEET.md
```
