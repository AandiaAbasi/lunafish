## 🔄 How to Update Documentation

If you need to modify documentation:

1. **Update API Code:** Modify `@extend_schema` decorators in `api/views.py`
2. **Update Guides:** Edit corresponding `.md` files
3. **Regenerate Schema:** Restart Django - schema auto-generates
4. **Verify:** Check https://fofofish.app/api/docs/swagger/
