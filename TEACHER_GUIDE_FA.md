# 👨‍🏫 راهنمای سیستم آزمون برای معلمان

سلام! این راهنما شما را در ایجاد و مدیریت آزمون‌های درسی اپ فوفوفیش راهنمایی می‌کند.

---

## 🎯 شروع سریع (10 دقیقه)

### مرحله 1️⃣: یک سؤال ایجاد کنید

1. بروید: **آزمون** → **سؤالات جدید**
2. **نوع سؤال را انتخاب کنید:**
   - `تایپی` - دانشآموز متن تایپ می‌کند
   - `تک گزینه‌ای` - دانشآموز یک گزینه انتخاب می‌کند
   - `چند گزینه‌ای` - دانشآموز چند گزینه انتخاب می‌کند
3. **عنوان سؤال** را وارد کنید
4. **توضیح یا راهنمایی** اضافه کنید (اختیاری)
5. **گزینه‌ها** را اضافه کنید (برای سؤالات چند گزینه‌ای)
6. **پاسخ صحیح** را مشخص کنید
7. **ذخیره** کنید

### مرحله 2️⃣: یک آزمون ایجاد کنید

1. بروید: **کلاس → آزمون‌های من**
2. روی **+ آزمون جدید** کلیک کنید
3. **سؤالات خود را انتخاب کنید**
4. **ترتیب سؤالات** را تنظیم کنید
5. **ذخیره و انتشار** کنید

### مرحله 3️⃣: نتایج دانشآموزان را ببینید

1. بروید: **نتایج**
2. **کلاس** را انتخاب کنید
3. **نمره‌های دانشآموزان** را ببینید
4. روی **هر دانشآموز** برای جزئیات

---

## 📝 انواع سؤالات

### ✏️ سؤالات تایپی

**متن:** دانشآموز پاسخ را تایپ می‌کند  
**نمره‌گذاری:** خودکار (دقیق)

**مثال:**
```
عنوان: نام پایتختِ فرانسه چیست؟
پاسخ انتظاری: پاریس
```

**کی استفاده کنید:**
- اسامی
- تاریخ‌ها
- تعاریف
- پاسخ‌های کوتاه

---

### 🔘 سؤالات تک گزینه‌ای (RadioButton)

**متن:** دانشآموز فقط یک گزینه انتخاب می‌کند  
**نمره‌گذاری:** خودکار (100%)

**مثال:**
```
عنوان: دو به اضافه دو برابر است با:
گزینه‌ها:
  ☐ 3 (غلط)
  ☑ 4 (صحیح)
  ☐ 5 (غلط)
```

**کی استفاده کنید:**
- مفاهیم
- تعریف‌ها
- انتخاب درست
- سؤالات معمولی

---

### ☑️ سؤالات چند گزینه‌ای (Checkbox)

**متن:** دانشآموز چند گزینه انتخاب می‌کند  
**نمره‌گذاری:** خودکار (تناسبی)

**مثال:**
```
عنوان: کدام مورد درست است؟
گزینه‌ها:
  ☑ فرانسه در اروپا است (صحیح)
  ☑ آلمان در اروپا است (صحیح)
  ☐ استرالیا در اروپا است (غلط)
```

**کی استفاده کنید:**
- سؤالات با چند پاسخ صحیح
- فهرست‌ها
- درست/غلط متعدد
- دسته‌بندی

---

## 🎯 ایجاد سؤالات خوب

### ✅ نکات بهتر

1. **سؤال را واضح بنویسید**
   - ❌ غلط: "چند است؟"
   - ✅ درست: "دو به اضافه دو برابر است با چند؟"

2. **گزینه‌های معقول اضافه کنید**
   - دقیقاً 3-5 گزینه
   - گزینه‌های نزدیک به هم
   - یک پاسخ صحیح واضح

3. **راهنمایی اضافه کنید**
   - راهنمایی برای پاسخ غلط
   - توضیح برای پاسخ صحیح
   - نکات مهم

4. **تصویر اضافه کنید**
   - تصویر برای سؤال
   - تصویر برای گزینه‌ها
   - نمودار یا جدول

## 📊 ایجاد آزمون

### مراحل آزمون

**Step 0 - بخش اول:**
```
سؤال 1، سؤال 2، سؤال 3
```

**Step 1 - بخش دوم:**
```
سؤال 4، سؤال 5
```

**Step 2 - بخش سوم:**
```
سؤال 6
```

### نکات

- **Step:** بخش‌های آزمون
- **Sort:** ترتیب سؤالات در هر بخش
- **نوع:** تایپی / تک گزینه / چند گزینه

**مثال:**
```
Step 0 (درک و یادگیری):
  - سؤال 1 (سؤال ساده)
  - سؤال 2 (تعریف)
  
Step 1 (کاربرد):
  - سؤال 3 (مسئله)
  - سؤال 4 (تحلیل)
```

## 📈 نمره‌گذاری

### نحوه محاسبه نمره

```
✅ پاسخ صحیح = 1 امتیاز
❌ پاسخ غلط = 0 امتیاز

نمره نهایی = (پاسخ صحیح / کل سؤالات) × 100
```

**مثال:**
- کل سؤالات: 5
- پاسخ صحیح: 4
- نمره: (4/5) × 100 = **80%**

### گزارش‌های نمره

دانشآموز می‌بیند:
- ✅ تعداد پاسخ‌های صحیح
- ❌ تعداد پاسخ‌های غلط
- 📊 درصد نهایی
- 🎯 نمره هر سؤال

## 🔌 اتصال به سیستم (Backend)

### چطور سیستم کار می‌کند؟

وقتی شما یک سؤال یا آزمون می‌سازید:

```
1. شما ➜ اپ موبایل
   (سؤال و گزینه‌ها را وارد می‌کنید)
        ⬇️
2. اپ ➜ سرور Backend
   (POST /api/exercise/field/create/)
        ⬇️
3. سرور ➜ Database
   (ذخیره سؤال و گزینه‌ها)
```

### مثال: ایجاد سؤال

**اپ شما این داده را می‌فرستد:**
```json
{
  "title": "پایتخت فرانسه کجاست؟",
  "type": "radioButton",
  "is_required": 1,
  "details": [
    {"title": "لندن", "is_correct": 0, "sort": 0},
    {"title": "پاریس", "is_correct": 1, "sort": 1},
    {"title": "برلین", "is_correct": 0, "sort": 2}
  ]
}
```

**سرور جواب می‌دهد:**
```json
{
  "data": {
    "id": 1,
    "title": "پایتخت فرانسه کجاست؟",
    "type": "radioButton",
    "details": [
      {"id": 1, "title": "لندن", "is_correct": 0},
      {"id": 2, "title": "پاریس", "is_correct": 1},
      {"id": 3, "title": "برلین", "is_correct": 0}
    ]
  },
  "message": "سؤال با موفقیت ایجاد شد"
}
```

### مثال: ایجاد آزمون

**اپ می‌فرستد:**
```json
{
  "teachingsubject_id": 5,
  "questions": [
    {"field_id": 1, "step": 0, "sort": 0},
    {"field_id": 2, "step": 0, "sort": 1},
    {"field_id": 3, "step": 0, "sort": 2}
  ]
}
```

**سرور جواب می‌دهد:**
```json
{
  "data": {
    "id": 5,
    "subject_id": 5,
    "subject_title": "جغرافیا 1",
    "total_questions": 3,
    "questions": [...]
  },
  "message": "آزمون با موفقیت ایجاد شد"
}
```

### توضیح Parameters

| نام | معنی | مثال |
|-----|------|------|
| field_id | ID سؤال | 1, 2, 3, ... |
| title | متن سؤال | "پایتخت فرانسه چیست؟" |
| type | نوع سؤال | radioButton, checkbox, input |
| is_correct | آیا گزینه صحیح است | 0 یا 1 |
| sort | ترتیب نمایش | 0, 1, 2, ... |
| is_required | الزامی بودن | 0 یا 1 |
| details | لیست گزینه‌ها | [...] |

### نکات فنی

✅ **درست:**
- یک سؤال، چند گزینه
- گزینه‌ها دارای ترتیب (sort)
- مشخص کردن گزینه صحیح
- متن راهنمایی برای هر سؤال

❌ **اشتباه:**
- فراموش کردن sort number
- چند گزینه صحیح بدون type=checkbox
- ذخیره سؤال بدون هیچ گزینه
